Parameters of the multi-scale GNNs with low-resolution graphs otained by Guillard's coarsening.
See Lino et al. (2022) (https://doi.org/10.1063/5.0097679).

The models included are:
    - NsTwoGuillardScaleGNN.chk. Load with: graphs4cfd.nn.mus_gnn.NsTwoScaleGNN.load_model('NsTwoGuillardScaleGNN.chk')
    - NsThreeGuillardScaleGNN.chk. Load with: graphs4cfd.nn.mus_gnn.NsThreeScaleGNN.load_model('NsThreeGuillardScaleGNN.chk')
    - NsFourGuillardScaleGNN.chk. Load with: graphs4cfd.nn.mus_gnn.NsFourScaleGNN.load_model('NsFourGuillardScaleGNN.chk')
